# Taska

Taska is a thoughtfully designed task management app that turns everyday productivity into a calm, structured, and visually engaging experience.

Built entirely with **HTML, CSS, and JavaScript**, it focuses on delivering a smooth and intuitive interface without relying on frameworks or backend systems.

---

## The Idea

Most task managers feel heavy, cluttered, or overwhelming.

Taska takes a different approach:
- keep the interface minimal  
- make interactions feel natural  
- give users clarity instead of complexity  

It’s not just about managing tasks — it’s about making the process feel effortless.

---

## What You Can Do

- Create, update, and remove tasks  
- Organize work using priority levels  
- Add due dates to stay on track  
- Use tags for flexible categorization  
- Instantly search through tasks  
- Switch between different views (all, active, completed, etc.)  
- Track overall progress visually  
- Toggle between light and dark modes  
- Keep data محفوظ (persisted) in the browser using LocalStorage  

---

## How It Works

Taska is a **frontend-only application**.

All task data is stored locally in the browser:
- no login  
- no database  
- no external dependencies  

This keeps the app:
- fast ⚡  
- lightweight  
- fully self-contained  

---

## Built With

- HTML — structure and layout  
- CSS — styling, layout system, and visual design  
- JavaScript — application logic and interactivity  

No frameworks. No libraries. Just core web fundamentals.

---

## Structure

```

.

````

The project is intentionally kept simple and readable, making it easy to understand and extend.

---

## Design Approach

The UI is designed around three principles:

**Clarity**  
Information is easy to scan and understand.

**Focus**  
No unnecessary distractions or clutter.

**Flow**  
Interactions feel smooth and predictable.
